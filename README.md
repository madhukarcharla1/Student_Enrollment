# Student_Enrollment
Assessment to create Angular UI for handling Student enrollment to few courses.

Clone the repository using https://github.com/madhukarcharla1/Student_Enrollment
run npm install
and then npm start; to run the app on the localhost:4200

Created Login component to ask for user name:  Any emailId and Any password works for trying the UI first.
Provided Students, Courses and Enrollments dashboard with tabbing.
Provided ability to add students, add courses and add enrollments to assign a course to student, and also provide grade.
Grade can be entered later on the enrollments for each student for each course.

Check the attached Screenshots.docx to go through the UI screens on the app.
